## 지침 우선순위 (Precedence)

충돌 시 상위 항목이 하위 항목을 오버라이드한다.

| 우선순위(내림차순) | 계층 | 설명 |
|----------|------|------|
| **1** | 프로젝트 헌법 | `CLAUDE.md`, `AGENTS.md` |
| **2** | 사용자 로컬 설정 | `.claude/`, `.codex/` |
| **3** | 세션 컨텍스트(사용자 입력) | 사용자 대화 중 지침 |
| **4** | 프로젝트 세부 명세 | `agent/` 하위 문서 |
| **5** | 에이전트 판단 | 상위 지침에 명시되지 않은 사항 |
